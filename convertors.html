Available Transformations

Choose from our collection of powerful text transformations. Each one is designed to save you time and improve your workflow.

Popular

Multiline to Single

Converts multiple lines of text into a single continuous line, removing line breaks.

Before:
Line 1
Line 2
Line 3
After:
Line 1 Line 2 Line 3
Perfect for cleaning up copied text

Single to Multiline

Splits a single line of text into multiple lines based on spaces or delimiters.

Before:
Word1 Word2 Word3
After:
Word1
Word2
Word3
Great for creating lists
Dev Tool

Pascal Case

Converts text to PascalCase format, capitalizing the first letter of each word.

Before:
hello world example
After:
HelloWorldExample
Essential for developers

Camel Case

Converts text to camelCase format, with first word lowercase and subsequent words capitalized.

Before:
hello world example
After:
helloWorldExample
JavaScript naming convention

Snake Case

Converts text to snake_case format, replacing spaces with underscores and lowercasing.

Before:
Hello World Example
After:
hello_world_example
Python naming convention

Kebab Case

Converts text to kebab-case format, replacing spaces with hyphens and lowercasing.

Before:
Hello World Example
After:
hello-world-example
Perfect for URLs and CSS
